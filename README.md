# JWT Auth RBAC Manager

A comprehensive authentication and role-based access control (RBAC) management system built with Spring Boot backend and React TypeScript frontend.

## Project Structure

- **Backend**: Spring Boot REST API with JWT authentication and RBAC
- **Frontend**: React + TypeScript with Vite, styled with Tailwind CSS

## Features

- JWT-based authentication
- Role-based access control (RBAC)
- Secure token management
- User registration and login
- Protected routes and components

## Tech Stack

### Backend
- Java
- Spring Boot
- Spring Security
- JWT (JSON Web Tokens)
- Maven

### Frontend
- React
- TypeScript
- Vite
- Tailwind CSS
- PostCSS

## Getting Started

### Prerequisites
- Java 11+
- Node.js 16+
- Maven

### Backend Setup
```bash
cd "JWTAuth  RBAC Manager/JWT-Auth-RBAC-Manager"
mvn clean install
mvn spring-boot:run
```

### Frontend Setup
```bash
cd JWTAuth-RBAC-Manager-UI
npm install
npm run dev
```
